## Требования

- Docker
- Docker Compose

## Установка и запуск

1. **Клонирование репозитория**:
```bash
git clone https://github.com/SANJIKS/Forms
```

2. **Переход в директорию проекта**:
```bash
cd Forms
```

3. **Построение образов с помощью Docker Compose**:
```bash
docker-compose build
```

4. **Запуск сервисов**:
```bash
docker-compose up
```
